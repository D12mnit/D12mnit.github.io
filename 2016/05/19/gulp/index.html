使用gulp构建项目

发表于 2016-05-19   |   分类于 前端   |   阅读次数

工作中从开发环境到生产环境，需要做许许多多的操作，诸如压缩图片，合并压缩js，css文件，生成css sprite图等等。如果持续靠人工来做这些事情真的是很无聊很烦躁，所以就要靠自动化工具来帮忙啦！

Gulp

Gulp.js是一个自动化构建工具，用来在开发项目的过程中自动执行一些固定的任务。类似的工具还有Grunt和Fis等。

Start

1.全局安装gulp-cli命令行工具

2.作为项目开发依赖安装gulp

3.在项目根目录下创建gulpfile.js文件

4.执行gulp任务

默认执行default任务，当然也可以通过$ gulp <task> <othertask>选择执行任务。
