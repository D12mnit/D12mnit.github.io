Flex布局语法笔记

发表于 2016-04-22   |   分类于 前端   |     |   阅读次数

谈到前端，当然离不开网页的布局，有别于传统的盒模型布局，Flex布局意为弹性伸缩布局，将会是未来布局的首选方案。废话不多说，先放兼容性列表！

可以看到目前主流浏览器都已经很好地支持了flex box，当然低版本的IE浏览器已经被排除在外，IE10+还是能很好的支持的。

对于移动端来说，webkit内核浏览器大都没什么问题，但是对于国产的一些奇葩浏览器比如UC，腾讯的X5内核等，则需要使用老版本的flex语法来兼容。在这里只想说，X5真是黑科技，我授予他响亮的称号：移动浏览器的IE！

废话不多说，我要开始装逼了。

开始使用Flex

既然也是盒模型，想要使用flex布局也很简单

或者是像行内元素一样使用

Flex模型


被设置成为flex布局的元素会成为flex容器（flex container），他的所有子元素都会自动成为flex元素（flex-item）。

flex容器默认拥有两条轴线：水平的主轴(main axis)与垂直的交叉轴(cross axis)；主轴的开始位置称为main start，结束位置称为main end，交叉轴开始位置称为cross start，结束位置称为cross end。

Flex 容器属性

Flex容器拥有以下属性设置

display

这个不用说了，设置容器为flex容器。注意，设置为flex容器的元素，css多列属性columns将会失效。

flex-direction

该属性定义了主轴方向。

  • row(default): 主轴为水平方向，从左到右排列。
  • row-reverse: 主轴为水平方向，从右到左排列。
  • column: 主轴为垂直方向，由上至下排列。

  • column-reverse: 主轴为垂直方向，由下至上排列。

    flex-wrap

    默认情况下，flex元素会默认排列在一条轴线上，该属性定义flex元素排列不下时如何换行。

  • nowrap(default): 不换行。

  • wrap: 换行，第一行位于上方。
  • wrap-reverse: 换行，第一行位于下方。

    flex-flow

    该属性为flex-direction与flex-wrap属性的简写形式。

justify-content

该属性定义了flex元素在主轴上的对齐方向。

  • flex-start(default): 主轴起点main start对齐。
  • flex-end: 主轴终点main end对齐。
  • center: 居中。
  • space-between: 两端对齐，且元素之间的间隙相等。
  • flex-end: 每个项目的两侧间隔相等并填满主轴方向，相当于每个元素的margin-left和margin-right相同。

align-items

该属性定义了flex元素在交叉轴上如何对齐。

  • flex-start: 交叉轴起点main start对齐。
  • flex-end: 交叉轴终点main end对齐。
  • center: 交叉轴中点对齐。
  • baseline: 元素的第一行文字的基线对齐。
  • stretch(default): 如果元素未设高度或设置为auto，则充满容器高度。

align-content

该属性仅在元素有多条轴线的情况下才会起作用，定义多轴线的对齐方式。

  • flex-start: 交叉轴起点main start对齐。
  • flex-end: 交叉轴终点main end对齐。
  • center: 交叉轴中点对齐。
  • space-between: 交叉轴两端对齐，元素之间间隔相等
  • space-around: 每根轴线的两侧间隔相等并填满交叉轴方向。
  • stretch(default): 轴线占满容器高度。
